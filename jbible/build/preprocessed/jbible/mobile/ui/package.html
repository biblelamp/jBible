Классы описания компонентов, относящихся к пользовательскому интерфейсу jBible.

Описывает компоненты, использующие высокоуровневый MIDP API, а также собственные компоненты, использующие низкоуровневый MIDP API. Данные компоненты реализуют также значительный объём функций по реализации обработки информации (поиск и т. п.)
