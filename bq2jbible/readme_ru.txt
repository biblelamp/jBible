BQ2jBible версия 1.10.2 от 24 апреля 2011г

1. Краткое описание

Программа разработана для конвертации модулей программы "Цитата из Библии" в модули для jБиблии. Написана на языке Java и представляет собой 1 (один) файл bq2jBible.jar размером 122'655 байт.

2. Требования

На вашем компьютере должна быть установлена Java Runtime Environment (JRE), которую можно загрузить по адресу http://java.sun.com/javase/downloads/

3. Как использовать

Создайте папку и скопируйте туда программу. Запустите программу. Нажмите на кнопку "Открыть", найдите папку с модулем к "Цитате из Библии" и выберите файл BIBLEQT.INI. В нижнем крупном окне программы появится список книг модуля. Отметьте книги, которые вы хотите включить в модуль jБиблии, дайте название модулю и нажмите кнопку "Записать". Программа создаст JAR файл в папке с соответствующим названием.

Модули к программе "Цитата из Библии" можно загрузить с сайта
http://bqt.ru/ или найти на компакт-диске "Библиология", который можно купить в онлайн-магазине - http://biblelamp.ru/eshop/catalog/?mc-cd144-1-
